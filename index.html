SVG Test

Dummy link above

A simple test to determine if SVG title is announced by screen readers in line-by-line mode.

Example 1: SVG inside a link

Documents 
<a href="#">
  Documents
  <span>
    <svg class="icon" version="1.1" viewBox="0 0 16 16">
      <title>Down arrow</title>
      <style>.svg1{fill:green;}</style>
      <polygon class="svg1" points="5,0 5,8 0,8 8,16 16,8 11,8 11,0" />
    </svg>
  </span>
</a>

Example 2: SVG on its own

<span>
  <svg class="icon" version="1.1" viewBox="0 0 16 16">
    <title>Up arrow</title>
    <style>.svg2{fill:red;}</style>
    <polygon class="svg2" points="11,16 11,8 16,8 8,0 0,8 5,8 5,16" />
  </svg>
</span>

Example 3: SVG on its own, with tabindex on parent

<span tabindex="0">
  <svg class="icon" version="1.1" viewBox="0 0 16 16">
    <title>Right arrow</title>
    <style>.svg3{fill:blue;}</style>
    <polygon class="svg3" points="0,11 8,11 8,16 16,8 8,0 8,5 0,5" />
  </svg>
</span>

Example 4: SVG on its own, with tabindex and aria-label on parent

<span tabindex="0" aria-label="Left arrow">
  <svg class="icon" version="1.1" viewBox="0 0 16 16">
    <title>Left arrow</title>
    <style>.svg4{fill:purple;}</style>
    <polygon class="svg4" points="16,5 8,5 8,0 0,8 8,16 8,11 16,11" />
  </svg>
</span>

Example 5: Hidden content outside SVG

Expand to open 
<span>
  <span class="hidden">Expand to open</span>
  <span aria-hidden="true">
    <svg class="icon" xmlns="http://www.w3.org/2000/svg" version="1.1" viewBox="0 0 16 16">
      <title>Expand</title>
      <style>.svg5{fill:black;}</style>
      <polygon class="svg5" points="16,5 11,5 11,0 5,0 5,5 0,5 0,11 5,11 5,16 11,16 11,11 16,11" />
    </svg>
  </span>
</span>
